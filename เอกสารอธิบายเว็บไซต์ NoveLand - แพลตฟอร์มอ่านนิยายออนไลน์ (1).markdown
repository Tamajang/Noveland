# เอกสารอธิบายเว็บไซต์ NoveLand - แพลตฟอร์มอ่านนิยายออนไลน์

## 📋 ภาพรวมของเว็บไซต์

**NoveLand** เป็นเว็บไซต์อ่านนิยายออนไลน์แบบ HTML/CSS/JavaScript ที่ออกแบบมาเพื่อให้ผู้ใช้สามารถค้นหา อ่าน และติดตามนิยายที่ชื่นชอบได้อย่างสะดวก โครงสร้างของเว็บไซต์นี้เน้นการใช้ HTML ที่เป็นโครงสร้างหลัก พร้อมการออกแบบ Dark/Light Mode และการตอบสนองต่ออุปกรณ์มือถือ

---

## 🏗️ โครงสร้างหลักของเว็บไซต์

### 1. **Navbar (แถบนำทางด้านบน)**
   - **ตำแหน่ง:** ติดอยู่ด้านบนของหน้าเว็บ (Fixed Position)
   - **ส่วนประกอบ:**
     - **Logo:** ชื่อเว็บ "NoveLand" ที่ใช้สีน้ำเงินไฟฟ้า (#405fff)
     - **เมนูนำทาง:** ลิงก์ไปยังหน้าต่างๆ (หน้าหลัก, ค้นหา, นิยายที่ติดตาม, กระเป๋าสตางค์, เติมเงิน)
     - **ปุ่ม Dark/Light Mode Toggle:** ปุ่มเปลี่ยนธีมสว่าง/มืด
   - **ฟังก์ชัน:** นำทางผู้ใช้ไปยังหน้าต่างๆ และจำเป็นต้องมองเห็นได้ตลอดเวลา
   - **ปรับตัว:** บน Desktop จะแสดงเมนูเต็ม บน Mobile จะซ่อนเมนูและแสดง Bottom Bar แทน

### 2. **Bottom Bar (แถบนำทางด้านล่าง - Mobile)**
   - **ตำแหน่ง:** ติดอยู่ด้านล่างของหน้าเว็บ (เฉพาะมือถือ)
   - **ส่วนประกอบ:**
     - ไอคอน 4 ปุ่ม: หน้าหลัก, ค้นหา, ติดตาม, ธีม
   - **ฟังก์ชัน:** ให้ผู้ใช้มือถือสามารถเข้าถึงฟังก์ชันหลักได้ง่าย
   - **ปรับตัว:** แสดงเฉพาะบนหน้าจอขนาด 768px ลงมา

## 📄 หน้าต่างๆ ของเว็บไซต์

### 1. **หน้าหลัก (index.html)**
   - **ส่วนประกอบหลัก:**
     - **Hero Section:** ส่วนแสดงภาพหลักและข้อมูลเด่น
     - **Carousel/Slider:** ภาพนิยายที่หมุนอัตโนมัติ
     - **Novel Cards:** การ์ดแสดงข้อมูลนิยายพร้อมรูปปก ชื่อ ผู้เขียน จำนวนผู้ชม และคะแนน
     - **Section ต่างๆ:**
       - "อัปเดตล่าสุด" - นิยายที่เพิ่งอัปเดตตอนใหม่
       - "ยอดนิยม" - นิยายที่มีผู้อ่านมากที่สุด
       - "Top 10 Ranking" - อันดับ 10 อันดับแรก
       - "แนะนำโดย NoveLand" - นิยายที่ระบบแนะนำ
       - "นิยายจบแล้ว" - นิยายที่จบเรื่องแล้ว
       - "นิยายทั้งหมด" - รายการนิยายทั้งหมด
   - **ฟังก์ชัน:** แสดงข้อมูลนิยายต่างๆ และให้ผู้ใช้สามารถคลิกเข้าไปอ่านได้

### 2. **หน้าค้นหา (search.html)**
   - **ส่วนประกอบ:**
     - **Search Input:** ช่องค้นหาชื่อนิยายหรือนักเขียน
     - **Filters (ตัวกรอง):**
       - หมวดหมู่ (แฟนตาซี, โรแมนติก, แอคชั่น, ลึกลับ)
       - สถานะ (กำลังดำเนินการ, จบแล้ว)
       - เรียงลำดับ (ล่าสุด, ยอดนิยม, คะแนนสูง, มีคนอ่านมาก)
     - **Novel Grid:** แสดงผลการค้นหาในรูปแบบกริด
   - **ฟังก์ชัน:** ให้ผู้ใช้ค้นหาและกรองนิยายตามเกณฑ์ต่างๆ

### 3. **หน้ารายละเอียดนิยาย (novel.html)**
   - **ส่วนประกอบ:**
     - **Novel Cover:** รูปปกนิยาย
     - **Novel Details:**
       - ชื่อนิยาย
       - ชื่อผู้เขียน
       - หมวดหมู่
       - สถานะ (กำลังดำเนินการ/จบแล้ว)
       - จำนวนผู้ชม
       - คะแนนรีวิว
       - คำอธิบายเรื่อง
       - ปุ่ม "เริ่มอ่าน"
     - **Chapter List:** รายการตอนทั้งหมดของนิยาย
   - **ฟังก์ชัน:** แสดงข้อมูลรายละเอียดของนิยายและให้ผู้ใช้เลือกตอนที่ต้องการอ่าน

### 4. **หน้าอ่านนิยาย (read.html)**
   - **ส่วนประกอบ:**
     - **Reading Bar:** แถบด้านบนแสดงชื่อนิยายและตอน พร้อมปุ่มตั้งค่า
     - **Settings Panel:** ส่วนควบคุมการอ่าน
       - ปรับขนาดตัวอักษร (A−, A+)
       - เปลี่ยนธีม (สว่าง, มืด, ซีเปีย)
       - ปรับระยะห่างบรรทัด (แคบ, ปกติ, กว้าง)
     - **Chapter Content:** เนื้อหาตอนนิยายที่อ่านได้
     - **Navigation Buttons:** ปุ่มนำทาง
       - ตอนก่อนหน้า
       - รายการตอน
       - ตอนถัดไป
   - **ฟังก์ชัน:** ให้ผู้ใช้อ่านนิยายด้วยความสะดวกสบาย พร้อมตัวเลือกการปรับแต่ง

### 5. **หน้าหมวดหมู่ (category.html)**
   - **ส่วนประกอบ:**
     - **Category Title:** ชื่อหมวดหมู่ (เช่น "หมวดหมู่: แฟนตาซี")
     - **Novel Grid:** นิยายในหมวดหมู่นั้นๆ
   - **ฟังก์ชัน:** แสดงนิยายตามหมวดหมู่ที่ผู้ใช้เลือก

### 6. **หน้านิยายที่ติดตาม (bookmarks.html)**
   - **ส่วนประกอบ:**
     - **Empty State:** ข้อความแสดงว่ายังไม่มีนิยายที่ติดตาม
     - **ปุ่มค้นหา:** ปุ่มเชื่อมไปยังหน้าค้นหา
   - **ฟังก์ชัน:** แสดงรายการนิยายที่ผู้ใช้ติดตาม (ในอนาคตจะต้องมี Backend)

### 7. **หน้ากระเป๋าสตางค์ (wallet.html) และ เติมเงิน (topup.html)**
   - **ฟังก์ชัน:** หน้าสำหรับการจัดการเงินและการเติมเหรียญ (ยังไม่พัฒนาเต็มที่)

## 🎨 ระบบ Dark/Light Mode

### วิธีการทำงาน:
1. **บันทึกการตั้งค่า:** ธีมที่ผู้ใช้เลือกจะถูกบันทึกใน `localStorage`
2. **การใช้ CSS Variables:** เว็บไซต์ใช้ CSS Custom Properties (--bg-dark, --bg-light, etc.) เพื่อเปลี่ยนสีตามธีม
3. **ปุ่ม Toggle:** ปุ่มเปลี่ยนธีมอยู่ที่ Navbar และ Bottom Bar

### สีที่ใช้:
- **Dark Mode:**
  - พื้นหลัง: #0a0a0a (ดำเข้ม)
  - ข้อความ: #e0e0e0 (เทาอ่อน)
  - Card: #1a1a1a
- **Light Mode:**
  - พื้นหลัง: #f5f8ff (ขาวอ่อน)
  - ข้อความ: #1a1a1a (ดำ)
  - Card: #ffffff (ขาว)
- **Primary Color:** #405fff (น้ำเงินไฟฟ้า) - ใช้สำหรับ Logo, Buttons, Links

## 📱 Responsive Design (ปรับตัวตามขนาดหน้าจอ)

### Desktop (> 768px):
- แสดง Navbar เต็ม พร้อมเมนูนำทาง
- ซ่อน Bottom Bar
- Novel Grid: 6-8 คอลัมน์

### Mobile (≤ 768px):
- ซ่อนเมนู Navbar Center
- แสดง Bottom Bar ด้านล่าง
- Novel Grid: 2-3 คอลัมน์
- ปรับขนาด Padding และ Font ให้เหมาะสม

## 💻 เทคโนโลยีที่ใช้

### HTML:
- **Semantic HTML:** ใช้ `<nav>`, `<main>`, `<article>`, `<section>` เพื่อให้โครงสร้างชัดเจน
- **Form Elements:** Input, Select สำหรับค้นหาและกรอง

### CSS:
- **CSS Variables (Custom Properties):** สำหรับจัดการสี และ Theme
- **Flexbox & Grid:** สำหรับ Layout
- **Media Queries:** สำหรับ Responsive Design
- **Transitions & Animations:** สำหรับ Hover Effects และ Smooth Changes

### JavaScript:
- **DOM Manipulation:** เปลี่ยนเนื้อหาตามการกระทำของผู้ใช้
- **LocalStorage:** บันทึกการตั้งค่า (Theme, Font Size, Line Height)
- **Event Listeners:** ตอบสนองต่อการคลิกปุ่มและการเลือก
- **URL Parameters:** ส่งข้อมูล Novel ID และ Chapter Number ผ่าน Query String

## 🔄 ขั้นตอนการใช้งาน (User Flow)

```
1. ผู้ใช้เข้าเว็บไซต์ → หน้าหลัก (index.html)
   ↓
2. ดูนิยายแนะนำหรือค้นหา → หน้าค้นหา (search.html)
   ↓
3. คลิกนิยายที่สนใจ → หน้ารายละเอียด (novel.html)
   ↓
4. เลือกตอนที่ต้องการอ่าน → หน้าอ่าน (read.html)
   ↓
5. ปรับตั้งค่าการอ่าน (ขนาดตัวอักษร, ธีม, ระยะห่างบรรทัด)
   ↓
6. อ่านนิยายและนำทางไปตอนถัดไป
```
